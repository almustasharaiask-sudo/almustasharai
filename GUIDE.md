# 🚀 المشترك AI - منصة الذكاء الاصطناعي المتقدمة

<div dir="rtl" align="right">

منصة احترافية وعصرية للذكاء الاصطناعي مع نظام مصادقة كامل وواجهة مستخدم متطورة.

## ✨ المميزات الرئيسية

- 🔐 **نظام مصادقة احترافي** - تسجيل دخول وإنشاء حسابات آمنة مع Supabase
- 🎨 **واجهة مستخدم حديثة** - تصميم احترافي جداً بتأثيرات متحركة
- 💬 **محادثات ذكية** - تفاعل طبيعي مع الذكاء الاصطناعي
- 📄 **معالجة الملفات** - رفع وتحليل ملفات متعددة الصيغ
- 👤 **لوحة تحكم المستخدم** - إدارة ملفك الشخصي والإعدادات
- 📊 **تحليلات متقدمة** - تقارير ورؤى عميقة
- 🌍 **دعم RTL كامل** - واجهة عربية احترافية

## 🛠️ التكنولوجيا المستخدمة

- **Framework**: Next.js 15
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Styling**: Tailwind CSS
- **Language**: TypeScript & JavaScript
- **AI**: Google Generative AI

## 📋 المتطلبات

- Node.js 18+
- npm أو yarn
- حساب Supabase (مجاني)

## 🚀 كيفية البدء

### 1. استنساخ المشروع
```bash
git clone https://github.com/almustasharaiask-sudo/almustasharai.git
cd almustasharai
```

### 2. تثبيت المكتبات
```bash
npm install
```

### 3. إعداد متغيرات البيئة
أنشئ ملف `.env.local` مع البيانات التالية:

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key

# AI
GEMINI_API_KEY=your_gemini_key

# Payment (اختياري)
PAYMENT_WALLET=your_wallet
PAYMENT_EMAIL=your_email
```

### 4. تشغيل السيرفر
```bash
npm run dev
```

السيرفر سيبدأ على `http://localhost:3000`

## 🎨 محتويات المشروع

```
src/

│   │   └── signup/
│   ├── dashboard/      # لوحة التحكم
│   ├── chat/          # صفحة المحادثات
│   ├── upload/        # صفحة رفع الملفات
│   ├── profile/       # الملف الشخصي
│   ├── features/      # صفحة المميزات
│   ├── about/         # عن المنصة
│   ├── page.js        # الصفحة الرئيسية
│   ├── layout.js      # التخطيط الرئيسي
│   └── globals.css    # الأنماط العامة
├── components/        # المكونات المشتركة
│   ├── Navbar.js      # شريط التنقل
│   ├── AnimatedCard.js
│   └── DashboardLayout.js
└── lib/              # المكتبات والدوال المساعدة
    ├── supabase.js   # إعدادات Supabase
    └── auth-context.js # Context للمصادقة
```

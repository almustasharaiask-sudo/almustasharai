# ⚖️ المستشار AI – منصة قانونية ذكية

أول منصة ذكاء اصطناعي متكاملة للقانون والائتمان في العالم العربي، تدعم 22 دولة و3 لغات.

## المميزات الرئيسية
- 5 شخصيات ذكاء اصطناعي: استشاري، دفاع، قاضي، محلل قانوني، محلل مالي
- مستشار فوري (شات) للإجابة على الأسئلة القانونية
- كاميرا OCR لتحليل المستندات المصورة
- رفع ملفات PDF و DOCX و TXT وصور
- قاعدة معرفة قانونية لكل دولة
- لوحة تحكم المالك (إدارة كاملة)
- نظام رصيد آمن مع سجل معاملات
- واتساب ديناميكي لطلب الشحن

## تشغيل المشروع
1. أنشئ مشروع Supabase جديد
2. نفذ SQL من ملف `supabase/migrations/001_initial.sql`
3. انسخ ملف `.env.local` وأضف مفاتيحك
4. `npm install`
5. `npm run dev`
